# Power Gym — Premium Fitness Center Website

A modern, responsive and visually impressive website for **Power Gym** — a premium fitness center located in Varna, Bulgaria.

---

## 🌐 Live Preview

Open `index.html` directly in any browser — no installation needed.

---

## 📋 Project Overview

This project was built as a real-world commercial website for a fitness brand. The goal was to create a professional, premium-looking website that could be used by an actual business — not a template or tutorial clone.

The website includes all the essential sections a modern fitness center needs: hero, about, services, team, membership plans, gallery, reviews, and a contact form.

---

## 🛠️ Technology Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox, grid, animations
- **Vanilla JavaScript** — scroll effects, counters, parallax, mobile menu
- **Google Fonts** — Bebas Neue, Barlow Condensed, Barlow
- **Font Awesome 6** — icons

> No frameworks, no build tools, no dependencies. Opens directly in the browser.

---

## 📁 Project Structure

```
power-gym/
│
├── index.html        # Main HTML file (self-contained)
└── README.md         # Project documentation
```

All CSS and JavaScript are embedded in `index.html` for portability. Images are base64-encoded and included inline.

---

## ✨ Features

- **Fully responsive** — works on mobile, tablet and desktop
- **Dark premium design** with lime/green neon accents matching the brand logo
- **Smooth animations** — fade-in on scroll, parallax hero, animated counters
- **Glassmorphism cards** for services, trainers and membership plans
- **Mobile navigation** with hamburger menu
- **Parallax hero** background with overlay
- **Animated counters** (500+ members, 10+ specialists, etc.)
- **Gallery section** with real gym interior photos and hover effects
- **Reviews section** with 5-star testimonials
- **Contact form** with interactive submit feedback

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/power-gym.git
   ```

2. Open `index.html` in your browser — that's it.

No npm, no build step, no server required.

## 🎨 Design Decisions

- **Color palette** derived from the Power Gym logo: dark backgrounds (`#090b09`) with lime green accents (`#84cc00`)
- **Typography**: Bebas Neue for headings (bold, impactful), Barlow Condensed for UI elements, Barlow for body text
- **Design language**: premium dark theme, subtle glow effects, glass morphism cards — inspired by high-end gym brands
